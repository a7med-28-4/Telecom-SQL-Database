Telecom Customer Database

This project is a SQL database design that simulates a telecom company system.
It was created to practice database design, DDL, relationships, and schema modeling using MySQL.

Database Overview

The database models a telecom system that manages:

Customers information

Subscription plans

Customer subscriptions

Call records

Customer churn status

The goal is to organize telecom data in a relational database structure and demonstrate how tables are connected using Primary Keys and Foreign Keys.
